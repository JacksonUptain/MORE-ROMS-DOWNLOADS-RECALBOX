=======================================================================
# RECALBOX: Sistema Amiga 1200 (Computer)
=======================================================================

Copia qui le rom per Amiga 1200.
I file supportati sono: .hdz .lzx .dms .ipf .adf .rp9 .adz .lha .zip .lhz .hdf .fdi .m3u

Questo sistema può essere emulato con 3 emulatori.
Tutti gli emulatori sono in questa lista, in ordine di compatibilità/semplicità di utilizzo:
amiberry-amiberry supporta i file di .adf .ipf .dms .fdi .hdf .hdz .lha .lhz .lzx .rp9 .zip e NON è compatibile con Netplay.
libretro-puae supporta i file di .adf .adz .dms .hdf .hdz .ipf .lha .lhz .lzx .m3u .zip e NON è compatibile con Netplay.
libretro-uae4arm supporta i file di .adf .adz .dms .hdf .hdz .ipf .lha .lhz .lzx .m3u .zip e NON è compatibile con Netplay.

Requisiti di sistema:
- Il Controller è obbligatorio
- La Tastiera è opzionale
- Il Mouse è opzionale

Questo sistema permette di utilizzare rom compresse in formato .zip o .7z.
Ma si tratta solamente di un file d'archivio. I file all'interno del .zip/.7z devono combaciare con le estensioni menzionate in precedenza.
Ogni .zip/.7z deve contenere solamente una rom compressa.
