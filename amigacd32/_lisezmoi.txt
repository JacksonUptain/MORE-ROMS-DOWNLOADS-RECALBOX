# RECALBOX: Système Amiga CD32 (Console de Salon)
=======================================================================

Copiez vos roms Amiga CD32 ici.
Les extensions supportées sont: .ccd .uae .mds .iso .m3u .cue .chd .nrg

Ce système peut être émulé par 3 émulateurs differents.
Ces emulateurs sont listés ci dessous par ordre de compatibilité/facilité d'utilisation :
amiberry-amiberry supporte des fichiers de type .cue .iso .ccd et n'est pas compatible netplay.
libretro-uae4arm supporte des fichiers de type .cue .ccd .chd .nrg .mds .iso .uae .m3u et n'est pas compatible netplay.
libretro-puae supporte des fichiers de type .cue .ccd .chd .nrg .mds .iso .uae .m3u et n'est pas compatible netplay.

Périphériques supportés :
- Les manettes sont obligatoires !
- Un clavier est optionel
- Une souris est optionelle


Consultez la documentation à cette adresse :
https://wiki.recalbox.com/fr/emulators/consoles/amiga-cd32
